Requirements for running the code:
1. Python 3
2. Libraries used:
	a. numpy.
	b. pandas.
	c. nltk
	d. sklearn
	e. scipy
	f. bs4
3. In line 127 and 129, change the path to the location 
	the DUC2001_Summarization_Documents/data/training directory is present.
4. In line 145 and 147, change the path to the location 
	where the DUC2001_Summarization_Documents/data/test/docs directory is present.
5. Run the code. All the accuracies and thresholds are printed for each classifier at each threshold.
